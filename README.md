# docai-project
DocAI – RAG-ассистент по PDF документам (web + n8n + Telegram)
Проект курса по вайб-кодингу.

## Состав

- `DocAI_new — active.html` – лендинг, UI, авторизация через Supabase, чат с документом.
- `My workflow 2.json` – схема n8n: upload-doc, chat, RAG, Telegram-бот, голосовой ввод.

## Как открыть

1. Скачать `DocAI_new — active.html` и открыть через локальный сервер (или GitHub Pages).
2. Импортировать `My workflow 2.json` в n8n (Workflows → Import from file).
